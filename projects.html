This website was created for an assignment in Back-End Development. However I decided to take it a step further by designing the front-end to follow the Spongebob theme. I also implemented many subtle animations using CSS such as the login form animation because I wanted the experience of the website to be smooth and natural.

I am proud of this project but there is some design errors that I created such as the background color of the games page. It should have been a white background so that the words can be clearly seen. Having the semi dark background drowns out the text which is bad for the user.
